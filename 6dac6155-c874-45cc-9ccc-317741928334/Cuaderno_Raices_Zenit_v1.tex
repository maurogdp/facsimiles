\begin{tcolorbox}[importantbox, title={¿Cómo usar este cuaderno?}]
Este cuaderno está diseñado para que resuelvas cada ejercicio directamente aquí, en el espacio asignado.

\medskip

\begin{itemize}
  \item Cada ejercicio tiene un \textbf{espacio de trabajo} delimitado por dos líneas horizontales. Úsalo para cálculos y procedimientos.
  \item Al terminar, escribe tu respuesta final en la línea \textbf{Respuesta:} al pie de cada ejercicio.
  \item Los bloques aumentan en dificultad. Si un ejercicio te cuesta, revisa los anteriores del mismo bloque.
  \item Las estrellas indican el nivel: $\bigstar$ básico · $\bigstar\bigstar$ intermedio · $\bigstar\bigstar\bigstar$ avanzado.
  \item El \textbf{Solucionario} al final tiene todas las respuestas. ¡Úsalo para verificar, no para copiar!
\end{itemize}
\end{tcolorbox}
